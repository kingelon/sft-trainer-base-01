# Lead Summary Component

> **Status:** 🔲 Not started

Generates a structured summary of a lead's profile for quick rep consumption.

## Planned Output

A concise lead summary covering:
- Company overview (industry, revenue, size)
- Current card/payment landscape
- Key signals and opportunities
- Recommended next steps

## Dependencies

- Shared input schema: [_schema/input_fields.md](../_schema/input_fields.md)
- May reference: [pitch/context/card_catalog.md](../pitch/context/card_catalog.md) for card knowledge
